### 🚀 De 0 a 100: JavaScript con G & Neith

"No memorizamos código. Entendemos cómo piensa JavaScript."

> Bienvenido.

Si estás leyendo este repositorio, probablemente quieras aprender JavaScript. Nosotros también.

La diferencia es que decidimos recorrer este camino de una forma distinta.

No queremos memorizar sintaxis ni copiar soluciones. Queremos entender cómo piensa JavaScript, por qué existe cada herramienta y cómo resolver problemas por nosotros mismos.

Utilizamos The Odin Project como guía del temario, pero cada explicación, cada proyecto y cada ejercicio han sido construidos por nosotros durante este recorrido.

Este repositorio no pretende mostrar código perfecto.

Pretende mostrar un aprendizaje real.

Aquí encontrarás errores, dudas, correcciones, proyectos, retos y muchas preguntas. Porque creemos que hacer preguntas es una de las mejores formas de aprender.

> 👋 Sobre nosotros

### Neith

Apasionado por la tecnología, la programación y la creación de proyectos con impacto social y cultural.

Su objetivo es convertirse en desarrollador Front-End y utilizar la programación para construir proyectos propios, aprender constantemente y abrir nuevas oportunidades profesionales.

### G

Profesor y compañero de aprendizaje durante este recorrido.

Mi objetivo es enseñar JavaScript desde los fundamentos, priorizando la comprensión sobre la memorización.

No busco enseñar únicamente qué escribir, sino ayudar a entender por qué el código funciona.

## 🎯 Nuestra meta

Más que terminar un curso, queremos desarrollar la capacidad de crear aplicaciones reales.

Cada clase está pensada para acercarnos un paso más al objetivo de conseguir un trabajo como desarrollador y, al mismo tiempo, construir una base que nos permita seguir aprendiendo durante muchos años.

## 📚 Nuestra metodología

Cada clase seguirá siempre la misma estructura:

- 📖 Comprender el concepto.
- 💡 Explicarlo con analogías.
- 💻 Escribir el código juntos.
- 🧪 Experimentar y romper el código para entenderlo.
- 🚀 Construir un proyecto.
- 🎯 Resolver un desafío.
- 📝 Documentar lo aprendido.
- 📁 Organización del repositorio

Cada clase tendrá su propia carpeta.

Clase-01/
│── index.html
│── style.css
│── script.js
│── README.md
│── Bitacora.md
│── Desafio.md
│── assets/

Queremos que este repositorio muestre no solo el resultado, sino también el proceso.

Por eso cada clase incluirá una bitácora con preguntas, respuestas, errores, descubrimientos y reflexiones.

### 🤝 Nuestro compromiso

Nos comprometemos a ser constantes.

A preguntar cuando algo no tenga sentido.

A celebrar los pequeños avances.

A no tener miedo de equivocarnos.

Y a recordar siempre que aprender programación no consiste en saber todas las respuestas, sino en desarrollar la capacidad de encontrarlas.

### 🌱 Una última reflexión

Hoy este repositorio contiene unas pocas líneas de texto.

Dentro de unos meses contendrá cientos de archivos.

Dentro de unos años esperamos que represente el comienzo de una carrera profesional.

> Y si algún día otra persona encuentra este repositorio y descubre que también puede aprender JavaScript desde cero, entonces todo este recorrido habrá valido todavía más la pena.
